# PokeMicro

A web project about consuming the Pokemon API. Allowing you to give likes to pokemons and add comments to each one.

1. Clone the repo

   ```sh
   git clone git@github.com:sevinchek/javascript-capstone.git
   ```
2. Install NPM packages

   ```sh
   npm install
   ```
